# Password Generator

A simple desktop password generator built with C# and WPF.

## About

This is my first C# project. Is it simple? Yes. Could you just use a website? Also yes. I just wanted to make this as a first project in C#.

## Features

- Generate passwords with customizable length
- Include/exclude character types:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!@#$%^&*()_+-=[]{}|;:,.<>?)
- Copy generated passwords to clipboard

## Requirements

- .NET 8.0 or later
- Windows (WPF application)

## How to Run

1. Clone this repository
2. Open `Password-Generator.sln` in Visual Studio
3. Build and run the project (F5)

## How to Use

1. Set your desired password length
2. Check/uncheck character type options
3. Click "Generate Password"
4. Click "Copy to Clipboard" to copy the password

## Technology Stack

- C# / .NET 8
- WPF (Windows Presentation Foundation)
- XAML for UI design
